blenderOSC
==========

Send and receive OSC message in Blender Game Engine with python 3.4,  easily, with examples.

#### Requirements

* python3.4 and more
* socket standard module
* pyOSCcodec: https://github.com/sergeLabo/pyOSCcodec
* blender 2.69 and more
